# FastFetch
My FastFech configs. These are written for my variety of systems, from laptops, desktops, and servers, and are geared towards a Linux environment.
I have also changed these slightly from any screenshots to make then distro-agnostic instead of forcing the openSUSE ASCII art.
They should work OOB on the type of systems they're noted for, but feel free to tweak them for proper display on w/e OS you'd like to use.

You can either overwrite your default config.jsonc or you can put this in your FF config folder:
~/.config/fastfetch/
and call them instead of your default with:
```
fastfetch -c <alternate_config_name>.jsonc
```

**Note: If you run one of these configs and the output is missing it's iconography, you are using a font that is missing the called-upon
glyphs. I recommend using a NerdFont in your VT to get the proper display.**
